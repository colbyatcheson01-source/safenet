# SafeNet — Product Hunt Launch Draft

## Tagline (60 char max)
Free online safety toolkit for parents — lessons, reporting, safety plans

## Description
SafeNet is a completely free, privacy-first toolkit that helps parents protect their kids online. No technical skills needed — download and run in 5 minutes.

**What it does:**
- 🎓 10 interactive safety lessons with quizzes for kids (privacy, phishing, grooming, cyberbullying, more)
- 🚨 One-click reporting tool for flagging harmful content to NCMEC / law enforcement
- 📝 Family safety plan generator — screen time rules, emergency contacts, step-by-step actions
- 📊 Parent dashboard with safety score, progress tracking, and daily tips
- 🔍 OSINT scanner — check if your child's username appears across 25+ platforms
- 🖱️ Chrome extension for right-click reporting from any page
- 🔄 Auto-updates — always has the latest features

**Why SafeNet is different:**
- Zero cost — pay what you can, if you can. No trials, no subscriptions, no data harvesting
- Runs locally on your computer — no cloud dependency, your data stays private
- Built for non-technical parents — download, double-click, done
- Open source — fully transparent about what it does

**Tech:**
- Standalone Windows EXE (40 MB) — no Python or dependencies needed
- Web version deploys anywhere (Railway, Render, VPS)
- Built with Python/Flask, SQLite
